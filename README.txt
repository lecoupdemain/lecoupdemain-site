# Contact via Formspree — Installation express
1) Crée un compte sur https://formspree.io et crée un **Form**.
2) Copie l’URL d’endpoint, elle ressemble à: `https://formspree.io/f/xyzabcde`.
3) Ouvre `contact.html` et **remplace** `TON_ENDPOINT` par cette URL.
4) Déploie les deux fichiers (`contact.html` et `success.html`) à la racine de ton site.
5) Teste: remplis le formulaire → tu dois recevoir l’email dans ta boîte pro.

Astuce anti-cache: fais Cmd+Shift+R (Mac) ou Ctrl+F5 (Windows) après remplacement.
